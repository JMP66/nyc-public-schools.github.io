Statistics for NYC Public Middle Schools

To give you an introduction to the public middle schools in New York we have gathered some key statistics that will be presented below.
In general, information about the five boroughs of New York, Bronx, Brooklyn, Manhattan, Queens and Staten Island is given. However, the first plot shows the total number of schools in each school district and which borough each school district belongs to, so you can track your favorite school district throughout the rest of the plots.
For a better overview, the color used for each borough is consistent throughout the plots.
If you use mouse hover on the plots the exact numbers will be presented.

Overview

On the top plot in the left hand side you can see how many public middle schools there are in each of the 32 school districts. Furthermore, the colors indicate which borough each of the districts belong to. The plot shows you the total number of schools in each borough. On the top plot in the right hand side you can see the total number of students enrolled in a public middle school in each of the boroughs. The plot below shows the average class size in each borough. This is calculated as the student-teacher ratio.
